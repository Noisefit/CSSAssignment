CSS can be added to HTML document in 3 Way:

  •  Inline - by  using the style attribute  inside HTML element1
  •  Internal - by using a style in the head section
  •  External - by using a link element to link an external CSS file.
  •  he most common way to add CSS, is to keep the styles in external CSs files. However,   in this tutorialwe will use inline and internal styles,because this is easier to demonstrate, and easier for you to try it yourself

When comparing CSS class vs ID,the difference is that CSS class applies a style to multiple elenments.ID,on the other hand, applies a style to one unique element.ID is also special in that you can use a special URl to link directly to an element and it's used by JavaScript.

In CSS,selectors are used to target a specific element or range of elenments on a web page.Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available.Two of the commonly used are class and ID.Both are used to target elements to which a style a style should be applied
